CSV Format Requirements

Your CSV should include these columns (column names are flexible):

  • sha256_hash or sha256 - File SHA256 hash
  • filename or description - File name/description
  • filepath or path - File path (optional)
  • file_type or type - MIME type (optional)
  • file_size or size - File size in bytes (optional)

Import Guidelines

Best Practices

  • • Use SHA256 hashes as unique identifiers
  • • Include descriptive filenames or descriptions
  • • Provide file paths when available
  • • Use standard MIME types for file_type
  • • Include file sizes in bytes

Important Notes

  • • Duplicate SHA256 hashes will be skipped
  • • Large imports may take several minutes
  • • Progress updates every 100 files
  • • Errors are logged for review
  • • All imports are tracked for auditing
